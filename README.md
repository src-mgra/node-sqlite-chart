# nodeSqliteChart
Example for
- reading data from file
- add data to sqlite-database
- show data in a chart

## cmd + URL
node addData2sqlite.js
node importFile event.txt
cmd: node showChart.js URL: http://localhost:3005

## needed packages
  "chart.js": "^2.7.2",
 
  "express": "^4.16.3",
  
  "sqlite3": "^4.0.2"
